# Оружие: нормализация урона и минимум 1 HP от взрывной волны

## Цель
- Каждое оружие в системе наносит урон (у всех `damage >= 1`).
- Если цель попала в радиус взрыва (любое оружие, которое вызывает `explodeAt`), то урон по червю всегда **минимум 1 HP**.

## Текущее поведение
В [PhysicsEngine.explodeAt](file:///workspace/src/presenters/PhysicsEngine.ts#L1580-L1657) урон вычисляется как:
- `damageRatio = 1 - dist/(finalRadius + playerRadius)`
- `actualDamage = cfg.damage * damageRatio`
Это может давать значения близкие к 0 на краю радиуса, и при округлении/порогах ощущается как “0 урона”.

## Решение
### 1) Минимальный урон от волны
В `explodeAt`:
- Если `dist <= finalRadius + playerRadius`, значит цель “задета волной”.
- Тогда:
  - `raw = cfg.damage * damageRatio`
  - применить `actualDamage = Math.max(1, raw)`
- Это гарантирует минимум 1 HP урона при любом задевании радиусом.

### 2) Нормализация параметров оружия
В [Weapon.ts](file:///workspace/src/models/Weapon.ts):
- Убедиться, что для всех оружий `damage >= 1`.
- Для “мелкого” оружия (minigun/heavy_gun) допускается `damage = 1`, но урон от взрыва/микро-взрывов теперь гарантированно не будет нулевым.

## Критерии готовности
- У всех записей `WEAPONS.*.damage >= 1`.
- При взрыве (любое оружие, вызывающее `explodeAt`) червь в радиусе всегда теряет минимум 1 HP.
- Все тесты проходят (`npm test`, `npm run build`).
