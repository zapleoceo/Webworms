# Техническое Задание (ТЗ) — WebWorms

## 1. Выполненные задачи (Текущий прогресс)
- **Спрайты и Анимация:** Интегрированы оригинальные покадровые спрайты (Worms, оружие, иконки). Внедрен `AnimationController` с автоматической очисткой фиолетового фона прозрачности. Оружие и точка выстрела корректно синхронизированы с вектором.
- **Улучшенная физика и коллизии:** Внедрен Bounding Box для червяков (анти-проскальзывание сквозь стены и мосты). Предметы при падении "впечатываются" (Stamp) в ландшафт и становятся разрушаемой текстурой.
- **Интерфейс и меню:** SVG заменены на оригинальные спрайты. Убрана лишняя панель Airdrop. Исправлен баг с пустым экраном при перезапуске игры.
- **Авторизация и профиль:** Настроена интеграция с Resend API для отправки писем подтверждения Email. В профиле добавлена форма для смены пароля.

## 2. План доработок (Новые требования)

### 2.1. Регистрация и Профиль
- **Смена пароля:** Убрать иконку "глазика" в поле нового пароля, пароль должен отображаться всегда явно (plain text).
- **Backend:** Реализовать и подключить API-эндпоинт для защищенной смены пароля пользователя в БД.

### 2.2. Главное меню и старт игры
- **Удаление старого функционала:** Полностью вырезать выбор класса червяка и выбор стартового оружия из главного меню.
- **Генерация команд:** При старте игры каждому игроку выдается команда из **3-х червяков**.
- **Случайный спавн:** Каждый червяк появляется в случайной безопасной точке на карте.
- **Случайное оружие:** Каждому червяку случайным образом навсегда присваивается **одно уникальное оружие** (базука, дробовик, и т.д.). Червяк будет использовать соответствующие этому оружию спрайты прицеливания.

### 2.3. Механика раунда и переключение червяков
- **Переключение червяков (Вместо оружия):** Механика смены оружия удаляется. Вместо нее вводится механика смены **активного червяка**.
- **UI команд:** Под игровой картой добавляется панель с 3 кнопками (иконка червяка + его HP) для управления своей командой. Нажатие делает червяка активным (им можно ходить и стрелять).
- **Смерть и Могилки:** Если здоровье червяка падает до 0, на его месте появляется спрайт могилки. Мертвый червяк исключается из переключения, не двигается и не стреляет.
- **Условие победы:** Раунд идет до тех пор, пока у одной из команд не погибнут все 3 червяка.
- **Конец игры:** По окончании матча кнопка возврата в меню будет выполнять полную перезагрузку страницы (`location.reload()`).

### 2.4. Таймер и Ходы
- **Движение до выстрела:** В свой ход игрок может переключиться на любого своего живого червяка, бегать им сколько угодно времени (в рамках таймера), и только после выстрела ход завершается.
- **Таймер хода:** По умолчанию 30 секунд. Если за 30 секунд игрок не сделал выстрел — ход сгорает и переходит к сопернику.
- **База данных (Админка):** Значение времени хода должно храниться в БД Cloudflare D1 и редактироваться через Админ-панель.
- **Оповещения (Тиканье):** Когда на таймере остается 5 секунд, он должен явно сигнализировать каждую секунду (визуально + звук тиканья).
- **Соло-режим (Training):** В режиме тренировки таймер хода становится **бесконечным**.
